# ДЗ №7: Операції з Двійковим Деревом Пошуку (BST)

Цей проєкт містить реалізацію Двійкового Дерева Пошуку (BST) та функції для виконання ключових операцій на ньому, відповідно до вимог домашнього завдання.

## Реалізовані Завдання

Усі функції працюють з властивостями BST, що забезпечує високу ефективність пошуку (зазвичай $O(h)$, де $h$ — висота дерева).

### 1. Знаходження Найбільшого Значення (Max)
**Алгоритм:** Ітеративно рухається від кореня **завжди праворуч** до останнього вузла.
* **Причина:** У BST всі значення у правому піддереві більші за значення батьківського вузла.

### 2. Знаходження Найменшого Значення (Min)
**Алгоритм:** Ітеративно рухається від кореня **завжди ліворуч** до останнього вузла.
* **Причина:** У BST всі значення у лівому піддереві менші за значення батьківського вузла.

### 3. Знаходження Суми Всіх Значень
**Алгоритм:** Рекурсивний обхід дерева (Depth-First Search, DFS), де на кожному кроці додається значення поточного вузла до суми результатів від лівого та правого піддерев.
* **Складність:** $O(n)$, оскільки необхідно відвідати кожен вузол дерева ($n$ — кількість вузлів).
